# Projeto-TCC-Senai
nosso repositorio tem como objetivo armazenar o desenvolvimento do nosso TCC

Escolhemos a demanda : Operação Enxuta na Cozinha: o dia em que o fluxo ‘cozinhou’ os custos

Descriçao:A empresa Kantine enfrenta falta de padronização operacional, controles de estoque frágeis e um layout que gera deslocamentos excessivos. Isso provoca furos de estoque, compras emergenciais, desperdícios de insumos, variação de qualidade e baixa previsibilidade de produção e compras, impactando diretamente o CMV e a produtividade da equipe.

Objetivos da soluçao :
- Reduzir o Custo de Mercadoria Vendida
- Diminuir compras emergenciais
- Melhorar o planejamento de produção 
- Implantar sistema de controle com Kanban
- Reduzir desperdícios de tempo e movimentação
- Melhorar o controle de estoque
- Padronizar processos com POPs e Fichas Técnicas
- Aumentar a produtividade sem necessidade de novas contratações

  Proposta de Solução

- Implantação de POPs
- Criação de fichas técnicas de produção
- Controle de estoque com planilhas automatizadas
- Aplicação do método PEPS 
- Uso da Curva ABC para priorização de insumos
- Planejamento semanal de produção 
- Sistema de reposição com Kanban 
- Melhoria do layout para reduzir deslocamentos 
- Aplicação de gestão visual

 Tecnologias Utilizadas

- Planilhas 
- Ferramentas de gestão visual
- Git e GitHub para versionamento


 Restrições do Projeto

- Baixo ou nenhum investimento financeiro
- Utilização da estrutura já existente
- Respeito às normas da Vigilância Sanitária (RDC 216)
- Soluções simples e práticas para uso no dia a dia
